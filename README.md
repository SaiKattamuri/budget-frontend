# Budget App - Frontend

Stack: React + Vite + TypeScript + Tailwind CSS + Recharts

## Quickstart

1. Install deps
```bash
npm install
```

2. Start dev server
```bash
npm run dev
```
Open http://localhost:5173

3. Environment
Set `VITE_API_URL` (defaults to http://localhost:8000/api)

## Build
```bash
npm run build
npm run preview
```
